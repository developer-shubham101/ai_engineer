# RAG Chat UI - Local Static Server Instructions

## Files
- `index.html`
- `css/styles.css`
- `js/app.js`

## How to run (Python)
1. From the project root (where `index.html` lives), run a simple Python static HTTP server:

**Python 3 (Linux / macOS / Windows with Python 3):**
```bash
python -m http.server 8080
```
